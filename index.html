Installation

Recommended

One-Liner Install

Installs uv and todopro-cli in one command:

curl -LsSf https://astral.sh/uv/install.sh | sh && \ uv tool install git+https://github.com/minhdqdev-org/todopro-cli.git

From Source

Clone and install locally:

git clone https://github.com/minhdqdev-org/todopro-cli.git cd todopro-cli uv tool install --from . todopro-cli

Update

Keep your installation up to date:

uv tool upgrade todopro-cli

Quick Start

Basic Usage

# Login to TodoPro todopro login # Quick add a task todopro add "Buy milk tomorrow at 2pm #groceries" # List tasks todopro tasks list # See what's due today todopro today # Get the next task to work on todopro next # Complete a task todopro complete <task_id>

Context Management

# Switch environments todopro config use-context prod # List available contexts todopro config get-contexts # Create a location context todopro contexts create home --icon "ğŸ " --geo # Check context availability todopro contexts check

Task ID Shortcuts

# Use suffixes instead of full IDs todopro complete abc123def # Full suffix todopro complete 123def # Short suffix todopro reschedule e562bb # Reschedule to today todopro get e562bb # Get task details todopro update 123def --content "Updated task"

Advanced Features

# Reschedule overdue tasks todopro tasks reschedule overdue --yes # Set timezone todopro auth timezone Asia/Ho_Chi_Minh # Output as JSON todopro tasks list -o json # Get help todopro --help
