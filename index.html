Features

CLI-First Design

Built for the command line from the ground up. Intuitive for humans, perfect for automation and AI agents.

Multi-Environment

Switch seamlessly between dev, staging, and prod contexts with separate credentials.

Natural Language

Create tasks with plain English like "Review PR tomorrow at 2pm #work @office p1"

Output Flexibility

Choose your format: JSON, YAML, table, or custom. Perfect for automation and scripting.

Professional UX

Rich terminal UI with colors, progress indicators, and helpful messages.

Context-Aware

Maintains authentication state and user preferences. Never lose your session.

End-to-End Encryption

Client-side encryption with AES-256-GCM. Your data is encrypted before it reaches the server. Zero-knowledge architecture.

AI-Agent Friendly

JSON output, semantic exit codes, and idempotent operations. Perfect for automation and AI workflows.

Import/Export

Easily backup and restore all your data. Export to JSON for portability and integration.

Installation

Recommended

One-Liner Install

Installs uv and todopro-cli in one command:

curl -LsSf https://astral.sh/uv/install.sh | sh && \ uv tool install git+https://github.com/minhdqdev-org/todopro-cli.git

From Source

Clone and install locally:

git clone https://github.com/minhdqdev-org/todopro-cli.git cd todopro-cli uv tool install --from . todopro-cli

Update

Keep your installation up to date:

uv tool upgrade todopro-cli

Quick Start

Basic Usage

# Login to TodoPro todopro login # Quick add a task todopro add "Buy milk tomorrow at 2pm #groceries" # List tasks todopro tasks list # See what's due today todopro today # Get the next task to work on todopro next # Complete a task todopro complete <task_id>

Context Management

# Switch environments todopro config use-context prod # List available contexts todopro config get-contexts # Create a location context todopro contexts create home --icon "üè†" --geo # Check context availability todopro contexts check

Task ID Shortcuts

# Use suffixes instead of full IDs todopro complete abc123def # Full suffix todopro complete 123def # Short suffix todopro reschedule e562bb # Reschedule to today todopro get e562bb # Get task details todopro update 123def --content "Updated task"

Advanced Features

# Reschedule overdue tasks todopro tasks reschedule overdue --yes # Set timezone todopro auth timezone Asia/Ho_Chi_Minh # Output as JSON todopro tasks list -o json # Get help todopro --help
